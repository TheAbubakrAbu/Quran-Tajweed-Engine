# Pre-computed tajweed annotations

This directory holds a **pre-computed tajweed layer** so any language can show colored tajweed without
porting the detection engine — just load the annotations and map each `rule` to a color from
[`../tajweed-rules.json`](../tajweed-rules.json).

## Files

- `NNN.json` — one file per surah (`001.json` … `114.json`), an array of ayah entries.
- [`../tajweed-annotations.json`](../tajweed-annotations.json) — the same data combined into one file (~5 MB).

## Format

```jsonc
[
  {
    "surah": 1,
    "ayah": 1,
    "annotations": [
      { "start": 0, "end": 1, "rule": "hamzatWaslSilent" },   // start/end = UTF-16 offsets
      { "start": 5, "end": 7, "rule": "tafkhim" }
    ]
  }
]
```

- `start` / `end` are **UTF-16 code-unit offsets** into that ayah's `textArabic` (Hafs) string, so
  `text.substring(start, end)` is the colored slice. (UTF-16 is what JavaScript, Swift `NSString`, Java,
  Kotlin, Dart, and C# use natively. In Python/Rust/Go, which index by code point or byte, convert — see
  the porting guide.)
- `rule` is a category id from `tajweed-rules.json` → look up `.categories[].colorHex`.
- Annotations are **non-overlapping** and in document order (the engine already resolved priority).

## Coverage

113,642 annotations across all 6236 ayahs (avg ~18 per ayah). Distribution: madd natural (34.9k),
tafkhim (21.3k), hamzat-wasl silent (13.3k), madd miniature (10.4k), ghunnah/ikhfaa/iqlaab families,
qalqalah (3.8k), and the madd muttasil/munfasil/lazim tiers.

## Provenance & caveats

Generated by the reference JS engine (`packages/quran-engine-js`, `detectPaintOps`) v0.1. It is a faithful
port of Al-Islam's heuristic tajweed engine with two documented simplifications (full final-`ر` vowel
context; muqatta'at lazim-harfi sub-typing) — see [../../docs/02-tajweed.md](../../docs/02-tajweed.md).
Treat this as a high-quality teaching/coloring layer, not a substitute for a certified mushaf. Regenerate
any time with:

```bash
node --input-type=module -e 'import { detectPaintOps } from "./packages/quran-engine-js/src/tajweed.js"; /* see scripts */'
```
